Опис:
 Програмне забезпечення призначене для безперервного спостереження за станом апаратних та програмних ресурсів комп'ютера. Головна мета — надати користувачу зручний інструмент для візуалізації навантаження на процесор, пам'ять, диски та мережу, а також забезпечити швидке виявлення "процесів-паразитів", які сповільнюють роботу ОС. Програма працює як інтерактивна панель керування, що дозволяє не лише бачити проблему, а й миттєво її вирішувати.
 
Можливості (фічі):

1. Динамічні графіки завантаження: Візуалізація використання CPU, RAM та швидкості обміну даними з диском у реальному часі за допомогою лінійних діаграм.
2. Інтерактивний менеджер процесів: Список усіх запущених програм із можливістю сортування за споживанням пам'яті або навантаженням на процесор.
3. Детектор "завислих" програм: Автоматичне підсвічування процесів, які мають статус "Not Responding" (не відповідає), для їх швидкого завершення.
4. Моніторинг температури: Відображення поточних показників температури ядер процесора та відеокарти (запобігання перегріву).
5. Мережевий сканер: Відстеження активних мережевих з'єднань: які програми зараз використовують інтернет та з якою швидкістю.
6. Система розумних сповіщень: Спливаючі повідомлення (Push-notifs), якщо критичний ресурс (наприклад, оперативна пам'ять) заповнений на понад 90%.
7. Аналіз здоров'я накопичувачів: Виведення інформації про вільне місце та S.M.A.R.T.-статус дисків (чи не збирається диск "померти").
8. Режим "Компактний віджет": Маленьке напівпрозоре віконце, що закріплюється поверх усіх програм і показує лише 2-3 головні цифри завантаження.
9. Моніторинг GPU та відеопам'яті: Спеціалізована панель для геймерів та дизайнерів. Вона відстежує завантаження графічного ядра, температуру відеокарти та об'єм зайнятої відеопам'яті (VRAM). Це допоможе виявити, чи не перевантажена система під час роботи з важкою графікою або іграми.
10. Експорт історії логів: Можливість зберегти звіт про стан системи за останні 24 години у форматі PDF або Excel для подальшого аналізу.
